# mdImages


快來看看全新的MIPS模擬器!
=========================

![](mdImages/mainScreen.png) \
\

專案介紹
--------


我們專案的目標是要製造出一個模擬32位元MIPS指令的模擬器，並實現先前可以執行MIPS指令的模擬器沒有做出的功能，即圖示化的Pipeline
Datapath。

支援的指令
----------

### R-type指令：

ADD、SUB、AND、OR、NOR、SLT、SLL、SRL

### I-type指令：

ADDI、LW、SW、ANDI、ORI、BEQ、BNE、SLTI

### J-type指令：

J
